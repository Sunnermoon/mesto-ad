# Проект Mesto

Учебный проект для Яндекс Практикума. Приложение для добавления и управления фотографиями мест с возможностью редактирования профиля, добавления новых карточек, лайков и удаления.

## Технологии

- **Vite** — сборщик проекта
- **JavaScript (ES6+)** — язык программирования
- **CSS3** — стилизация
- **API Mesto** — сервер для хранения данных

## Установка и запуск

### Установка зависимостей
```bash
npm install
```

### Запуск локального сервера разработки
```bash
npm run dev
```
Сервер запустится на `http://localhost:5173/`

### Сборка проекта
```bash
npm run build
```
Собранный проект появится в папке `dist/`

### Публикация на GitHub Pages
```bash
npm run deploy
```

## Структура проекта

```
src/
├── scripts/
│   ├── components/
│   │   ├── api.js          — функции для работы с API
│   │   ├── card.js         — функции для работы с карточками
│   │   ├── modal.js        — функции для работы с модальными окнами
│   │   └── validation.js   — функции для валидации форм
│   └── index.js            — точка входа приложения
├── pages/
│   └── index.css           — главный файл стилей
├── blocks/                 — CSS блоки по методологии БЭМ
├── images/                 — изображения проекта
└── vendor/                 — внешние библиотеки и шрифты
```

## Функциональность

- ✅ Загрузка данных пользователя и карточек с сервера
- ✅ Редактирование профиля (имя и описание)
- ✅ Обновление аватара пользователя
- ✅ Добавление новых карточек
- ✅ Удаление карточек (только для автора)
- ✅ Лайки карточек
- ✅ Просмотр полноразмерного изображения
- ✅ Валидация всех форм
- ✅ Индикаторы загрузки при отправке данных
